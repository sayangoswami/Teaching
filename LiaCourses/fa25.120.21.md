#### The ATM Withdrawal

Enhance the `BankAccount` class from before.

- **Method:** A **public** method `withdraw(double amount)` that returns a **boolean**.
    
- **Logic:** The withdrawal must only happen if two conditions are met: 1) `amount` > 0 AND 2) `balance` $\ge$ `amount`.
    
- If the withdrawal succeeds, update `balance` and return `true`. Otherwise, return `false`.

# Task Manager (To-Do List)

**Description:** Create a personal task manager to keep track of things you need to do. The application should allow you to add new tasks and see a dashboard overview of your productivity.

**Core Components to Use:**

- **"Add Task" Page (`Form`):** A form to add a new task.
    
    - **Fields:** `TaskName` (String), `Priority` (String - "High", "Medium", "Low"), `DueDate` (String), `Status` (String - "Pending").
        
- **"All Tasks" Page (`Table`):** A table that shows a list of all tasks.
    
- **"Dashboard" Page (`Panel`):** A main dashboard page that shows:
    
    - **`StatCard` 1:** Total Tasks Pending.
        
    - **`StatCard` 2:** Total High Priority Tasks.
        
    - **`Chart` 1:** A **Pie Chart** showing tasks broken down by `Priority`.
        
    - **`Chart` 2:** A **Bar Chart** showing tasks broken down by `Status` (e.g., "Pending", "In Progress", "Done" - _see bonus features_).
        

**Key OOP Requirements:**

1. **`Task` Class:** You must create a `Task` class that implements the `Rowable` interface. It will store the data for a single task (name, priority, etc.).
    
2. **`AddTaskHandler` Class:** You must create a class that implements `FormSubmitHandler` to process the data from your `Form`, create a new `Task` object, and add it to your main `ArrayList` of tasks.
    
3. **Dynamic Dashboard:** The `StatCard`s and `Chart`s on your dashboard must be _dynamic_. They must read their data from your `ArrayList` of tasks and update automatically as new tasks are added.

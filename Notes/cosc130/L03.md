# More on Arrays


## Recap

To declare an array variable:
```java
int[] scores;
```

To create an empty array and store in an array-variable

```java
scores = new int[25];
```

To do both in the same line
```java
int [] scores
scores = new int[25];
```



## Recap

To access (read/write) an array element:
```java
int i = 10;
scores[i] = scores[i-1] + scores[i+1];
```


## Initializing an array with values

Syntax:
```java
type[] array_name = { value0, value1, value2, â€¦ , valueM };
```

Example:
```java
int[] arr = {5, 10, 15, 20, 25};
for (int i = 0; i < 5; i++)
	System.out.println(arr[i]);
```

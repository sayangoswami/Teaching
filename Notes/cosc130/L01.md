# Refresher - Variables, Operators, Branches


## Agenda

1. Variables - declaration, assignment
2. Operators - arithmetic, logical, string
3. Expressions
4. Conditional Statements



## Declaring variables

A *variable declaration* is a *statement* that r*eserves a named space* in memory to *store a value*. The syntax is to specify the data type followed by the variable name.

**Syntax:** 
```
dataType variableName;
```


### Examples

1. **Integer:**
    
```java
int studentCount;
```
    
2. **String (text):**
    
```java
String welcomeMessage;
```
    
3. **Decimal Number:**
    
```java
double averageScore;
```
    
4. **Boolean (true/false):**
    
```java
boolean isComplete;
```


## Assigning values

The *assignment statement* places a specific piece *of data into a variable*. The assignment operator (`=`) is used for this purpose. You can assign a value at the time of declaration or later in your code.

**Syntax:** 
```
variableName = value;
```


### Examples

1. **Combined Declaration and Assignment:**
    
```java
int numStudents = 25;
```
    
2. **Separate Declaration and Assignment:**
    
```java
String courseName;
courseName = "Computer Science II";
```
    
3. **Assigning a decimal value:**
    
```java
double courseGrade = 94.5;
```
    
4. **Assigning a boolean value:**
    
```java
boolean passed = true;
```


## Arithmetic operators

These operators are used to perform mathematical calculations.

|Operator|Name|Example|
|---|---|---|
|`+`|Addition|`a + b`|
|`-`|Subtraction|`a - b`|
|`*`|Multiplication|`a * b`|
|`/`|Division|`a / b`|
|`%`|Modulus|`a % b`|

The *Modulus* operator is used to *find the remainder*. We will use this often.


### Examples

1. **Addition and Subtraction:**
    
```java
int sum = 10 + 5;      // sum is 15
int difference = 20 - 7; // difference is 13
```
    
2. **Multiplication and Division:**
    
```java
double product = 4.5 * 2.0; // product is 9.0
double quotient = 10.0 / 4.0; // quotient is 2.5
```
    
3. **Integer Division:**
    
```java
int result = 10 / 4;    // result is 2 (the decimal part is truncated)
```
    
4. **Modulus (Remainder):**
    
```java
int remainder = 10 % 3; // remainder is 1
```


## Expressions

An *expression* is a *combination of variables, values, and operators* that *evaluates to a single value*. Expressions can be placed on the right-hand side of an assignment statement.

**Syntax:** 
```
dataType variable = expression;
```


### Examples

1. **Simple Arithmetic Expression:**
    
```java
int finalScore = midTermScore + finalExamScore;
```
    
2. **Expression with multiple operators:**
    
```java
double total = (item1Price + item2Price) * salesTax;
```
    
3. **Expression with a method call:**
    
```java
double circumference = 2 * Math.PI * radius;
```
    
4. **Boolean expression:**
    
```java
boolean isPassing = (score > 70);
```


## String operators

The primary operator for strings is the `+` operator, which performs concatenation. It combines two or more strings to create a new one.

**Syntax:** 
```java
String newString = string1 + string2;
```

### Examples

1. **Combining two string literals:**
    
```java
String greeting = "Hello" + " World!"; // greeting is "Hello World!"
```
    
2. **Combining a string variable and a literal:**
    
```java
String userName = "Jane";
String welcomeMessage = "Welcome, " + userName; // welcomeMessage is "Welcome, Jane"
```


### Examples (contd.)

3. **Concatenating strings with numbers:**
    
```java
int numItems = 3;
String message = "You have " + numItems + " items in your cart.";
// message is "You have 3 items in your cart."
```
    
4. **Building a path:**
    
```java
String root = "/users/";
String path = root + userName + "/documents";
// path is "/users/Jane/documents"
```


## Printing variables

You can display the value stored in a variable using `System.out.println()`. The `+` operator is used to concatenate (combine) strings with variables to create a more readable output.

**Syntax:** 
```java
System.out.println(variableName);
System.out.println("Text" + variableName);
```


### Examples

1. **Printing an integer:**
    
```java
int score = 85;
System.out.println(score);
// Output: 85
```
    
2. **Printing with a descriptive string:**
    
```java
String student = "Alice";
System.out.println("The student is: " + student);
// Output: The student is: Alice
```


### Examples (contd.)

3. **Printing multiple variables:**
    
```java
int count = 5;
String item = "books";
System.out.println("You have " + count + " " + item + ".");
// Output: You have 5 books.
```
    
4. **Printing the result of a calculation:**
    
```java
double total = 15.75;
System.out.println("The total is $" + total);
// Output: The total is $15.75
```


## Relational operators

These operators are used to compare two values. The result of a relational operation is always a `boolean` value (`true` or `false`).

| Operator | Name                     | Example  |
| -------- | ------------------------ | -------- |
| `==`     | Equal to                 | `a == b` |
| `!=`     | Not equal to             | `a != b` |
| `<`      | Less than                | `a < b`  |
| `>`      | Greater than             | `a > b`  |
| `<=`     | Less than or equal to    | `a <= b` |
| `>=`     | Greater than or equal to | `a >= b` |

### Examples

1. **Equality check:**
    
```java
int x = 10;
int y = 5;
boolean isEqual = (x == y); // isEqual is false
```
    
2. **Greater than check:**
    
```java
boolean isPositive = (x > 0); // isPositive is true
```


### Examples (contd.)

3. **Not equal to check:**
    
```java
String name1 = "Bob";
String name2 = "bob";
boolean notEqual = (name1 != name2); // notEqual is true
```
    
4. **Less than or equal to check:**
    
```java
int age = 16;
boolean canDrive = (age >= 16); // canDrive is true
```


## Logical operators

These operators are used to combine `boolean` expressions.

| Operator | Name | Description                           |
| -------- | ---- | ------------------------------------- |
| `&&`     | AND  | `true` if both conditions are `true`  |
| `\|\|`   | OR   | `true` if either conditions is `true` |
| `!`      | NOT  | Inverts the boolean value             |

### Examples

1. **AND (`&&`):**
    
```java
boolean isEligible = (age > 18) && (hasLicense);
// true only if age is > 18 AND hasLicense is true
```
    
2. **OR (`||`):**
    
```java
boolean isStudent = (isUndergrad || isGradStudent);
// true if either isUndergrad or isGradStudent (or both) are true
```
    
3. **NOT (`!`):**
    
```java
boolean isFinished = false;
if (!isFinished) {
	// This code will run
}
```


### Examples (contd.)

4. **Combining operators:**
    
```java
boolean canEnter = (hasTicket && isAdult) || isVIP;
// true if you have a ticket AND are an adult OR if you are a VIP
```


## Boolean variables

A `boolean` variable can only hold one of two values: `true` or `false`. They are essential for storing the result of logical and relational expressions and for controlling the flow of a program.

### Examples

1. **Storing a simple condition:**
    
```java
int temperature = 25;
boolean isWarm = temperature > 20; // isWarm is true
```
    
2. **Storing a complex logical expression:**
    
```java
boolean canRentCar = (age >= 25) && (hasValidLicense);
```


### Examples (contd.)

3. **Used as a flag to track state:**
    
```java
boolean hasSaved = false;
// ... code ...
hasSaved = true; // State is updated
```
    
4. **Directly assigning a value:**
    
```java
boolean isRunning = true;
```


## Conditional statements

The `if` statement allows a program to execute a block of code only if a certain condition is met. The `if-else` statement provides an alternative block of code to run if the condition is false.

**Syntax:**

```java
if (condition) {
    // code to run if condition is true
} else {
    // code to run if condition is false
}
```

### Examples

1. **Simple `if` statement:**
    
```java
if (score >= 90) {
	System.out.println("You got an A!");
}
```
    
2. **`if-else` statement:**
    
```java
if (age >= 18) {
	System.out.println("You are an adult.");
} else {
	System.out.println("You are not an adult yet.");
}
```


### Examples (contd.)

3. **Checking for a specific value:**
    
```java
String day = "Tuesday";
if (day.equals("Monday")) {
	System.out.println("It's the start of the week.");
} else {
	System.out.println("It's not Monday.");
}
```
    
4. **Using a boolean variable:**
    
```java
boolean isRaining = true;
if (isRaining) {
	System.out.println("Grab an umbrella.");
}
```

## Chaining conditional statements

The `if-else if-else` chain allows you to check for multiple, mutually exclusive conditions. The computer will evaluate each `if` and `else if` in order, and will execute the first block of code whose condition is `true`, then skip the rest.

**Syntax:**

```java
if (condition1) {
    // Code if condition1 is true
} else if (condition2) {
    // Code if condition2 is true
} else {
    // Code if all conditions are false
}
```


### Examples

1. **Categorizing numbers:**
    
```java
int num = 0;
if (num > 0) {
	System.out.println("Positive");
} else if (num < 0) {
	System.out.println("Negative");
} else {
	System.out.println("Zero");
}
```


### Examples (contd.)

2. **Time of day:**
    
```java
int hour = 14; // 2 PM
if (hour < 12) {
	System.out.println("Morning");
} else if (hour < 18) {
	System.out.println("Afternoon");
} else {
	System.out.println("Evening");
}
```


### Examples (contd.)

3. **Traffic light status:**
    
```java
String light = "yellow";
if (light.equals("red")) {
	System.out.println("Stop");
} else if (light.equals("yellow")) {
	System.out.println("Slow down");
} else if (light.equals("green")) {
	System.out.println("Go");
} else {
	System.out.println("Invalid input");
}
```

## Nesting conditional statements

Nesting is the practice of placing one conditional statement inside another. This allows you to check for a second condition only after a primary condition has already been met.

**Syntax:**

```java
if (outerCondition) {
    // Code that only runs if outerCondition is true
    if (innerCondition) {
        // Code that only runs if both are true
    }
}
```

### Examples

1. **Access check:**
    
```java
boolean isLoggedIn = true;
boolean isAdmin = true;
if (isLoggedIn) {
	if (isAdmin) {
		System.out.println("Access granted to admin panel.");
	}
}
```


### Examples (contd.)

2. **Combination with `else`:**
    
```java
boolean hasTicket = true;
int age = 22;
if (hasTicket) {
	if (age >= 18) {
		System.out.println("Entry permitted.");
	} else {
		System.out.println("Ticket is valid, but age requirement not met.");
	}
} else {
	System.out.println("No ticket found.");
}
```


### Examples (contd.)

3. **Weather conditions:**
    
```java
boolean isRaining = true;
boolean isWindy = true;
if (isRaining) {
	System.out.println("It's raining.");
	if (isWindy) {
		System.out.println("Bring a heavy coat!");
	} else {
		System.out.println("Just a light rain.");
	}
}
```


### Examples (contd.)

4. **Complex user validation:**
    
```java
String username = "user123";
String password = "password123";
if (username.length() > 5) {
	if (password.length() > 8) {
		System.out.println("User credentials are valid.");
	} else {
		System.out.println("Password is too short.");
	}
} else {
	System.out.println("Username is too short.");
}
```

## Taking user input with `Scanner`

In Java, getting user input is like using a special tool called a **`Scanner`**. This tool knows how to "scan" what a person types on the keyboard.

To use this tool, you need to do two things:

1) **Get the tool ready:** We have to tell our program that we're going to use this **`Scanner`** tool. We do this by writing a special line at the very top of our code.
	```java
	import java.util.Scanner;
	```

2) **Turn the tool on:** Inside our main program, we create the tool and tell it to look at the keyboard. We'll always use this exact line for now.
	```java
	Scanner input = new Scanner(System.in);
	```

Here, **`input`** is a variable that stores the tool. For now, think of **`input`** as a name we're giving our tool so we can *call* it later.


## Asking Questions and Getting Answers

Once you have your `Scanner` tool (`input`), you can use it to ask for different kinds of information. We'll use special commands to tell the `Scanner` what to look for.

|Command|What It Gets|Example|
|---|---|---|
|**`input.nextInt()`**|A whole number (like 10 or 500)|`int age = input.nextInt();`|
|**`input.nextDouble()`**|A number with a decimal (like 3.14)|`double price = input.nextDouble();`|
|**`input.nextLine()`**|A whole line of text (a sentence)|`String name = input.nextLine();`|
